## Parent guide: understanding curriculum progress

- **Review the overall status badge** to see whether the child is ahead, on track, or behind.  
- **Inspect core vs. extension skill rows** to identify specific gaps (e.g., times tables, doubles, division).  
- **Use the recommended focus actions** to immediately start a practice session from the curriculum card. These actions pre-configure the right topic type and skill domain.  
- **Revisit the selector** if your child changes school placement, curriculum, or grade levelâ€”saving a new year/grade resets the benchmark and recalculates progress.

## In-app help & accessibility

- Every curriculum panel includes tooltip text (hover or focus the info icon) that reiterates what each badge means and how to adjust the year/grade.  
- The Year/Grade selector is built with keyboard focus management, high-contrast tokens, and touch-friendly buttons so the flow remains accessible (see `design-documentation/accessibility/guidelines.md`).  
- All recommendations and status indicators expose text alternatives so screen readers announce the skill label, proficiency, and recommended action.

If you still have questions, tap the **Help & support** icon inside the grown-ups tab to open live chat or send feedback through the in-app contact form.
