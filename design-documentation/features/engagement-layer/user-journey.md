# Engagement Layer – User Journey

## During Play

- Small, non-blocking affirmations appear for streaks or milestones.

## After Round

- Summary surfaces any new badges, with a brief celebratory animation.

## Longer-Term

- Progress view highlights ongoing streaks or badges earned over time.
