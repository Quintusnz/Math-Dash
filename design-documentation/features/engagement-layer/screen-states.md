# Engagement Layer – Screen States

## In-Game Indicators

- Small, unobtrusive streak indicator near top or side.

## Post-Game Celebration

- Confetti or banner showing achievement unlocked, using `long` duration and skippable behavior.

## Ongoing Overview

- Progress or profile screens show streaks and badges in a compact strip.
