# Progress Dashboard & Skill Radar – Screen States

## Child Progress View

- Default with data: shows streak, stars, recent badges, and simple per-topic cards.
- Empty: encouraging message and suggestion to play a first dash.

## Adult Progress View

- Topic list with proficiency bands and sparkline trends.
- Optional radar/spider chart summarizing topics.

## Topic Detail View

- Shows accuracy and speed trends for one topic.
