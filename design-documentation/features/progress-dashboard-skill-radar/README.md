# Progress Dashboard & Skill Radar – Overview

## Overview

This feature provides visual summaries of a child’s progress across topics, exposing simple, encouraging views for children and more detailed diagnostics for adults.

## Primary User Goals

- Children feel proud of visible progress (stars, streaks, badges).
- Parents/teachers quickly see strengths and gaps by topic.

## Success Criteria

- Child-facing views emphasize celebration and simple trends.
- Adult-facing views show per-topic proficiency and recent activity.
