# Progress Dashboard & Skill Radar – User Journey

## Child Journey (In-App)

1. From Play Home, tap `My Progress`.
2. Sees a simple summary: streaks, stars, recent achievements.
3. May tap a topic to see a simple trend (e.g., “You’re getting faster!”).

## Adult Journey (Grown-Ups Area)

1. In Grown-Ups area, choose a child.
2. Sees topic-level proficiency view and trend over time.
3. Can filter by time range (e.g., last week, last month).

## From Round Summary

- After a game, parent/child can tap `See more progress` to open the dashboard focused on that topic.
