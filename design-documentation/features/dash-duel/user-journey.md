# Dash Duel – User Journey

## Setup

1. From Play Home or Grown-Ups area, select `Dash Duel`.
2. Choose Player 1 and Player 2 profiles (or guest).
3. Select shared topic and mode.
4. Start duel.

## During Duel

- Split-screen layout; each player answers their own stream of questions.

## Summary

- Shows each player’s score and a friendly winner message.
- Offers quick rematch option.
