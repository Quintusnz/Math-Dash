# Dash Duel – Overview

## Overview

Dash Duel is a same-device two-player mode where players race to answer their own questions and compare results at the end.

## Primary User Goals

- Children enjoy friendly competition with siblings or friends.
- Adults can set up a fair duel configuration.

## Success Criteria

- Clear visual separation between Player 1 and Player 2 areas.
- Summary clearly shows who won while staying positive.
