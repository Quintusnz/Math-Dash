# Feature Documentation Index

This directory contains detailed design specifications for each major feature of Math Dash. Each feature folder includes user journeys, screen states, interaction details, accessibility guidelines, and implementation notes.

## Core Experience

- **[Core Dash Round Gameplay](./core-dash-round-gameplay/README.md)**  
  The primary single-player game loop: timed or count-based math rounds with immediate feedback.

- **[Topic & Skill Library](./topic-and-skill-library/README.md)**  
  The catalogue of math topics (tables, bonds, etc.) and the system for tracking proficiency per topic.

- **[Game Modes](./game-modes/README.md)**  
  Configuration for different ways to play: 30s/60s Dash, 10/20 Questions, and Practice.

## Progression & Identity

- **[Player Profiles & Avatars](./player-profiles-and-avatars/README.md)**  
  Multi-profile management allowing siblings to share a device with distinct progress and avatars.

- **[Progress Dashboard & Skill Radar](./progress-dashboard-skill-radar/README.md)**  
  Visualizations of progress: simple celebrations for children, detailed skill radars for adults.

- **[Engagement Layer](./engagement-layer/README.md)**  
  Gamification elements including streaks, badges, and celebratory moments to drive retention.

## Advanced Features

- **[Adaptive Practice Mode](./adaptive-practice-mode/README.md)**  
  A smart practice mode that adjusts difficulty and topic selection based on real-time performance.

- **[Dash Duel](./dash-duel/README.md)**  
  Local split-screen multiplayer mode for head-to-head math races on a single device.

## System & Business

- **[Monetization & Access Control](./monetization-and-access-control/README.md)**  
  Freemium logic, premium content locking, and adult-gated upgrade flows.

- **[Accounts & Cloud Sync](./accounts-and-cloud-sync/README.md)**  
  Systems for data persistence, cross-device synchronization, and account management.
