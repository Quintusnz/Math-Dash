# Player Profiles & Avatars – Screen States

## Profile Picker

- Default: shows a card for each child profile: avatar, name, short status.
- Empty: friendly empty state prompt with `Add a player` CTA.

## Profile Creation

- Step 1: Name & age range fields.
- Step 2: Avatar selection grid with clear selection state.

## Profile Edit

- Similar to creation, but with existing values prefilled.
- Archive/delete option visually separated and clearly marked.
