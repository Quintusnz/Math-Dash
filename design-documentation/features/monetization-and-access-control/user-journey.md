# Monetization & Access Control – User Journey

## Child Encountering Locks

- Child taps a locked topic or mode.
- Sees a gentle, non-pressuring message that this area is for grown-ups.

## Parent Upgrade Flow

1. From grown-ups area or lock prompt, parent passes adult gate.
2. Sees clear explanation of upgrade benefits and price.
3. Completes upgrade using platform-appropriate purchase flow.
4. Returns to app; premium content is now unlocked.
