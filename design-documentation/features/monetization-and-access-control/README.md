# Monetization & Access Control – Overview

## Overview

This feature defines how free vs paid content is represented in the UI, and how upgrades are initiated and enforced in a child-safe way.

## Primary User Goals

- Parents clearly understand what is free vs paid and why.
- Children are not pestered or confused by monetization.

## Success Criteria

- Clear lock visuals on premium topics and modes.
- Upgrade flows always sit behind adult gate.
