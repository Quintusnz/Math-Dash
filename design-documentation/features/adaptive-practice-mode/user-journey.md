# Adaptive Practice Mode – User Journey

## Starting Adaptive Practice

1. From Play Home, child selects `Practice` mode.
2. Topic can be fixed or mixed, depending on configuration.
3. Child taps `Start practice`.

## During Practice

- No visible timer.
- Difficulty and topic selection adjust quietly based on recent answers.

## Ending Practice

- Child taps `End practice`.
- Summary shows encouraging message and may suggest a focus topic.
